# Community Listing Templates

Use these snippets when submitting `grok-agent-kit` to npm, MCP communities, or skills collections.

## Short subtitle

`CLI + MCP + skills for xAI Grok`

## One-line description

`grok-agent-kit` is a local-first toolkit that gives Codex, Claude Code, OpenClaw, and terminal users the same Grok-powered chat and search workflows.

## Key points

- Local-first CLI and stdio MCP server
- `XAI_API_KEY` as the primary runtime auth path
- Optional `XAI_MANAGEMENT_API_KEY` for local team API key management
- Shared guidance for Codex, Claude Code, and OpenClaw
- Bilingual docs for English and Simplified Chinese users
